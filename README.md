# SuperHeroesSDP

SuperHeroesSDP is a fun and visually appealing app built with SwiftUI, where you can explore a catalog of original superheroes along with their stories and powers. It's ideal for experimenting with SwiftUI and custom model management.

## Features
- Visual superhero grid
- Cards with detailed information (name, nickname, description, age, powers, image)
- Easy navigation using SwiftUI NavigationStack
- Reusable components (information rows, grid cells)
- Adaptive and modern interface

## Getting Started

### Prerequisites
- Xcode 26 or later
- iOS 26 or later

### Installation
1. Clone the repository:
```bash
git clone https://github.com/nacher8/SuperHeroesSDP.git
```

2. Open the project in Xcode:
```bash
cd SuperHeroesSDP
open SuperHeroesSDP.xcodeproj
```

3. Build and run the project on your simulator or device.

## Technologies Used
